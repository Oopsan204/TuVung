Hướng dẫn sử dụng

Đồng bộ với GitHub Gist:

Xác thực với GitHub và tạo/cập nhật Gist để lưu trữ dữ liệu từ vựng của bạn. Bạn có thể truy cập dữ liệu này từ bất kỳ thiết bị nào bằng cách nhập cùng token và Gist ID.

Sao lưu dữ liệu:

Sử dụng nút "Xuất dữ liệu" để tạo bản sao lưu cục bộ dưới dạng file JSON. Bạn có thể khôi phục dữ liệu bằng nút "Nhập dữ liệu".

Cài đặt ứng dụng

Nhập GitHub Personal Access Token

Để sử dụng tính năng đồng bộ với GitHub Gist, bạn cần tạo và nhập Personal Access Token.

Hướng dẫn tạo GitHub Token:

  1. Truy cập GitHub Settings → Developer settings → Personal access tokens (fine-grained)
  2. Nhấp "Generate new token"
  3. Đặt tên cho token và chọn thời hạn
  4. Quan trọng: Trong phần "Repository access", chọn "Public Repositories (read-only)" HOẶC "All repositories"
  5. Trong phần "Account permissions", chọn "Gists" với quyền "Write"
  6. Nhấp "Generate token" và copy token
  7. Dán token vào ô trên

Lưu ý: Token chỉ hiển thị 1 lần, hãy lưu lại để sử dụng sau.
